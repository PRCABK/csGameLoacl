COMBAT FEEL

打起来有反馈，看起来也清楚

战斗反馈

准星变色、Hitmarker、击杀横幅、命中闪白、低血量脉冲，每一次交火都更有爽感。

CS 风雷达

圆形小地图显示朝向扇形、发现敌人、开火暴露；团队模式队友常显。

低模战术风

统一调色板与材质分级，角色阵营染色 + 头顶标识，远处也能快速辨敌友。

即开即玩

Node 一键启动，浏览器直连。适合宿舍、公司、朋友局临时开黑。

GAME MODES

两种模式，一种节奏

大厅先选模式，再选地图。房间按“模式 + 地图”隔离，没人自动关闭。

DEATHMATCH

死斗

所有玩家互为敌人。拼走位、拼预瞄、拼击杀榜。

  • 个人击杀统计
  • Tab 全员计分板
  • 10 分钟战局结算
TEAM DEATHMATCH

团队竞技

红蓝对攻，同队不可误伤。比的是阵线与配合。

  • 红蓝分侧出生
  • 顶部队伍总分
  • Tab 分队击杀/死亡

BATTLEFIELDS

货运仓库

QUICK START

本地启动，朋友连入

terminal 
git clone https://github.com/PRCABK/csGameLoacl.git
cd csGameLoacl
npm install
npm start

然后打开 http://localhost:3000

01

主机启动服务

运行 npm start，确认本机可访问 3000 端口。

02

查局域网 IP

Windows 执行 ipconfig，拿到主机 IPv4 地址。

03

朋友连入开黑

同网段访问 http://主机IP:3000，选模式与地图即可。
